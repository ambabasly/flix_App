Welcome!

The purpose of this project is to create the server-side component of a "movies" web application using JavaScript, Node.js and Express. The web application will provide users with access to information about various movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Business Logic URL HTTP Method Request body data format Response body data format
Get a list of movies /movies GET None A JSON object holding data about all the movies
Get data about a single movie by name /movies/[Name] GET None A JSON object holding data about a single movie,
Example:
{
id: 1,
title: "The Way Back",
year: 2020
}
Add a movie /movies POST A JSON object holding data about the movie to add,
Example:
{
title: "365 Days",
year: 2020
} 		A JSON object holding data about the movie that was added, incuding an ID:
{
id: 5,
title: "The Way Back",
year: 2020
}
Remove a movie by ID /movies/[ID] DELETE None A text message indicating whether the movie was successfully removed
Update the year of a certain movie /movies/[Title]/[year] PUT None A text message indicating whether the year of the certain movie was updated
